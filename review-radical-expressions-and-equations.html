    Section 4.7 Radical Expressions and Equations Chapter Review

    Square and \(n\)th Root Properties.

    The square root of some number is a non-negative number you can multiply with itself to get that first number. For example, the square root of \(9\) is \(3\) because \(3\cdot3=9\text{.}\) The symbol for a square root uses a radical, like \(\sqrt{9}\text{.}\)
    üîó
    Memorizing some square roots can be helpful.
    üîó
    \(\sqrt{0}=0\) \(\sqrt{1}=1\) \(\sqrt{4}=2\) \(\sqrt{9}=3\)
    \(\sqrt{16}=4\) \(\sqrt{25}=5\) \(\sqrt{36}=6\) \(\sqrt{49}=7\)
    \(\sqrt{64}=8\) \(\sqrt{81}=9\) \(\sqrt{100}=10\) \(\sqrt{121}=11\)
    \(\sqrt{144}=12\) \(\sqrt{225}=15\) \(\sqrt{256}=16\) \(\sqrt{400}=20\)
    Many square roots have decimal expressions that go on forever, like \(\sqrt{2}=1.4142\ldots\text{.}\) A calculator can provide these decimal expressions or you can estimate them.
    üîó
    There are also higher index roots, also known as \(n\)th roots. The \(n\)th root of some number is a number you can multiply with itself \(n\) times to get that first number. The \(n\)th root of a positive number is positive. The \(n\)th root of a negative number is negative when \(n\) is odd, and does not exist when \(n\) is even. The symbol for an \(n\)th root uses a radical with \(n\) written in the pocket, like \(\sqrt[3]{8}\text{.}\)
    üîó
    The \(n\)th root of a number \(a\) is a solution to the equation \(x^n=a\text{.}\)
    üîó
    Square roots and \(n\)th roots have algebra properties that can be used to simply radical expressions.
    Root of a Product Property
    üîó
    \(\displaystyle \sqrt[m]{a\cdot b} = \sqrt[m]{a} \cdot \sqrt[m]{b}\)
    üîó
    Root of a Quotient Property
    üîó
    \(\sqrt[m]{\frac{a}{b}} = \frac{\sqrt[m]{a}}{\sqrt[m]{b}}\) as long as \(b\neq 0\)
    üîó
    üîó
    A goal is to simplify any radical expression such that the radicand (the number inside the radical) is a whole number, but as small as possible. This can be done by ‚Äúfactoring‚Äù the radicand into factors where one facotr is a perfect square, and then using the first property above.
    üîó
    More complicated radical expressions with more than one radical might be simplified using radical properties and combining like radical terms.
    üîó
    üîó

    Rationalizing the Denominator.

    For various reasons, a radical inside a denominator in undesirable. The same is true for a denominator inside a radical. Here are some examples of these undesirable expressions.
    \begin{equation*} \frac{3}{\sqrt2}\quad\frac{1+\sqrt{3}}{4\sqrt{5}}\quad\frac{7}{3+\sqrt{5}}\quad\sqrt{\frac35} \end{equation*}
    There are algebra steps that can be taken to write numbers like these without a radical in the denominator or a denominator inside a radical. The process is called ‚Äúrationalizing the denominator‚Äù.
    üîó
    If the denominator is a single term with a radical, multiplying the entire expression by that radical over itself will not change the value of the expression, but it will leave you with a rational denominator. For example:
    \begin{equation*} \frac{3}{\sqrt2}\cdot\frac{\sqrt2}{\sqrt2}=\frac{3\sqrt{2}}{2}\qquad\frac{1+\sqrt{3}}{4\sqrt{5}}\cdot\frac{\sqrt5}{\sqrt5}=\frac{\sqrt{5}+\sqrt{15}}{20} \end{equation*}
    üîó
    If the denominator has two terms where one or both are radicals, multiplying the entire expression by the conjugate of that denominator over itself will not change the value of the expression, but it will leave you with a rational denominator. For example:
    \begin{equation*} \frac{7}{3+\sqrt{5}}\cdot{3-\sqrt{5}}{3-\sqrt{5}}=\frac{14-7\sqrt{5}}{3^2-5}=\frac{14-7\sqrt{5}}{4} \end{equation*}
    üîó
    If there is a fraction within that denominator, you can use a property of radicals to separate the numerator and denominator into their own radicals:
    \begin{equation*} \sqrt{\frac35}=\frac{\sqrt3}{\sqrt5} \end{equation*}
    And then you can do as we did above in the first example:
    \begin{equation*} \frac{\sqrt3}{\sqrt5}\cdot\frac{\sqrt5}{\sqrt5}=\frac{\sqrt15}{5} \end{equation*}
    üîó
    üîó
    Simplify each expression. If the result has a denominator, it must be rationalized.
    üîó

    \(\displaystyle{ \frac{4}{\sqrt{2}} }\)
    üîó
    Explanation.

    üîó

    üîó

    üîó

    üîó

    Radical Expressions and Rational Exponents.

    The square root of a number is the same thing as raising that number to the \(\frac12\) power. That is, \(\sqrt{x}=x^{1/2}\text{.}\) Similarly, the \(n\)th root of a number is the same thing as raising that number to the \(\frac1n\) power. That is, \(\sqrt[n]{x}=x^{1/n}\text{.}\) More generally,
    \begin{equation*} x^{m/n}=\sqrt[n]{x^m}=\left(\sqrt[n]{x}\right)^m \end{equation*}
    üîó
    When given radical expressions, using the above to write them with rational exponents can be useful for simplifying the expressions. This is because we have all of the exponent properties from Section¬†2 and other places. So we have a lot of tools that work well for expressions with exponents.
    üîó
    So one possible approach to simplifying a radical expression is to convert it to use rational exponents and then see what you can do using exponent properties. After some progress has been made, you might also convert your result back into a radical form.
    üîó
    üîó

    Solving Radical Equations.

    If an equation has one variable that appears one time in the equation, and it is within a radical, then solving the equation is relatively straightforward. First isolate the radical. Then raise each side of the equation to the appropriate power to undo the radical. This either solves the equation for you or leaves you with only some basic steps left to apply.
    üîó
    üîó
    Solve each equation.
    üîó

    Checkpoint 4.7.10.

    \(\displaystyle{ {\sqrt{3t}} = {9} }\)

    \(\displaystyle{ {3\sqrt{4-t}+9} = {39} }\)
    üîó

    To solve an equation with radicals, we first separate the radical on one side of the equation, and then square both sides:
    üîó
    \(\displaystyle{\begin{aligned} {3\sqrt{4-t}+9} \amp = {39} \\ {3\sqrt{4-t}+9} \mathbf{{}-9} \amp = {39} \mathbf{{}-9} \\ 3\sqrt{4-t} \amp = {30} \\ \frac{3\sqrt{4-t}}{3} \amp = \frac{{30}}{3} \\ \sqrt{4-t} \amp = 10 \\ (\sqrt{4-t})^2 \amp = (10)^2 \\ 4-t \amp = 100 \\ 4-t \mathbf{{}-4} \amp = 100 \mathbf{{}-4} \\ -t \amp = 96 \\ t \amp = -96 \end{aligned} }\)

    Checkpoint 4.7.12.

    \(\displaystyle{ \sqrt[3]{x-8} = {-8} }\)
    üîó
    Explanation.
    To get rid of the cube root in this equation, we raise the left side and the right side of the equation each to the third power.
    üîó
    \(\displaystyle{\begin{aligned} \sqrt[3]{x-8} \amp = {-8} \\ \left(\sqrt[3]{x-8}\right)^3 \amp = \left(-8\right)^3 \\ x-8 \amp = -512 \\ x-8+8 \amp = -512+8 \\ x \amp = {-504} \\ \end{aligned} }\)
    üîó
    We must check our solutions when we solve radical equations. We therefore substitute our answer into the original equation:
    üîó
    \(\displaystyle{\begin{aligned} \sqrt[3]{x-8} \amp \stackrel{?}{=} {-8} \\ \sqrt[3]{{-504}-8} \amp \stackrel{?}{=} -8 \\ \sqrt[3]{} \amp \stackrel{?}{=} -8 \\ -8 \amp \stackrel{?}{=} -8 \\ \end{aligned} }\)
    üîó
    That is true, so the solution set is \(\{{-504}\}\text{.}\)
