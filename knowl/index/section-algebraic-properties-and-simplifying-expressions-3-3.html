We start with some definitions. The number \(0\) is called the additive identity. It has this name because adding \(0\) to a number does not change that number’s “identity”. If you were playing a “game” where you needed to add something to \(x\text{,}\) but you didn’t want to change the value, then you would add \(0\text{:}\)
\begin{equation*} x+0=x \end{equation*}
Adding the additive identity to a number does not change that number.
