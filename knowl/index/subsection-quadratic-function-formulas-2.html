A function \(f\) is a quadratic function if it has a formula in the form
\begin{equation*} f(x) = ax^2 + bx + c \end{equation*}
for some numbers \(a\text{,}\) \(b\text{,}\) and \(c\text{.}\) While \(b\) and \(c\) are allowed to be \(0\text{,}\) the number \(a\) must not be \(0\) (or else we would just have a linear function).
