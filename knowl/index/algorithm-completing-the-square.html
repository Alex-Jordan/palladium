Process 2.8.4. Completing the Square.

For a quadratic equation simplified to the form \(x^2+bx=c\text{,}\) to solve for \(x\) by completing the square,
  1. Use Fact 3 to find the number to add to both sides of the equation to make the left hand side a perfect square. This number is always \(\highlight{\left(\frac{b}{2}\right)^2}\text{.}\)
    🔗
    🔗
  2. Add that number to both sides of \(x^2+bx=c\) to get
    \begin{equation*} x^2+bx\addright{\left(\frac{b}{2}\right)^2}=c\addright{\left(\frac{b}{2}\right)^2} \end{equation*}
    🔗
    🔗
  3. The left hand side is now a perfect square that factors as \(x^2+bx+\left(\frac{b}{2}\right)^2=\left(x+\frac{b}{2}\right)^2\text{,}\) so the equation becomes
    \begin{equation*} \left(x+\frac{b}{2}\right)^2=c+\left(\frac{b}{2}\right)^2 \end{equation*}
    🔗
    🔗
  4. Solve remaining equation using the Square Root Property.
    🔗
