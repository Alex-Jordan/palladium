Definition A.5.10. Interval Notation.

Interval notation describes an interval by saying where it “starts” and “ends”. It can look like any of these four options:
\begin{equation*} (\text{start},\text{end})\qquad (\text{start},\text{end}]\qquad [\text{start},\text{end})\qquad [\text{start},\text{end}] \end{equation*}
