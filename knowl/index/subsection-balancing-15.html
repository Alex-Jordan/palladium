In theory, there could have been more than one solution to the equation (although that is not the case with this equation). So conceptually, there is a collection of solutions to any given equation. We call this collection a solution set. Any set of numbers that only has one or a few numbers in it is written using curly braces. In this case, the solution set is \(\{4\}\text{.}\) Using braces to surround a collection of numbers listed out is called set notation, not to be confused with set-builder notation from Section 5.
