    üîó
    Previous several sections covered methods for plotting a graph of a linear equation in two variables.
    üîó
    We‚Äôve learned three forms for a linear equation:
    üîó
    slope-intercept form \(y=mx+b\)
    üîó
    point-slope form \(y=m\left(x-x_0\right)+y_0\)
    üîó
    standard form \(Ax+By=C\)
    üîó
    We‚Äôve studied two special types of line:
    üîó
    horizontal line \(y=k\)
    üîó
    vertical line \(x=h\)
    üîó
    We‚Äôve practiced three ways to graph a line:
    üîó
    building a table of \(x\)- and \(y\)-values
    üîó
    plotting one point (often the \(y\)-intercept) and drawing slope triangles
    üîó
    plotting its \(x\)-intercept and \(y\)-intercept
    üîó
    In this section, we review these ideas, especially what steps to take to plot a graph of a linear equation.
    üîó

    Subsection 2.9.1 Graphing Lines in Slope-Intercept Form

    To review various methods for plotting a line, we‚Äôll start by taking one line, \(y=-2x+1\text{,}\) and graph it using each of the strategies we‚Äôve learned.
    üîó

    Example 2.9.2. Building a Table.

    First, we‚Äôll graph \(y=-2x+1\) by building a table of values. Our first task is to pick some \(x\)-values to use in the table, and we arbitrarily choose \(-2,-1,0,1,2\text{.}\) It‚Äôs a good habit to use at least five values when plotting this way. After this, we carefully evaluate \(y\) over and over, and plot the resulting points.
    üîó
    üîó

    Example 2.9.5. Using Slope Triangles.

    Although making a table is straightforward, the slope triangle method is faster and reinforces the true meaning of slope. With the slope triangle method, we first identify some point on the line. Our line is in slope-intercept form, so we know the \(y\)-intercept is \((0,1)\text{.}\) Plot this first, and then we can draw slope triangles in both directions to find more points. The slope we will use is also apparent: \(-2=\frac{-2}{1}\text{.}\)
    üîó
    Compared to the table method, the slope triangle method:
    • is more sophisticated;
      üîó
    • doesn‚Äôt take the time and space to make a table;
      üîó
    • doesn‚Äôt involve lots of calculations where you might make a human error;
      üîó
    • shows slope triangles, which reinforces the meaning of slope.
      üîó
    üîó
    üîó

    Example 2.9.8. Using intercepts.

    If we use \(x\)- and \(y\)-intercepts to plot \(y=-2x+1\text{,}\) we have to find those intercepts first. For our current line, it is apparent that the \(y\)-intercept is at \((0,1)\text{.}\) But where is the \(x\)-intercept?
    üîó
    Set \(y=0\text{.}\)
    \begin{align*} y\amp=-2x+1\\ \substitute{0}\amp=-2x+1\\ 0\subtractright{1}\amp=-2x\\ -1\amp=-2x\\ \divideunder{-1}{-2}\amp=x\\ \frac{1}{2}\amp=x \end{align*}
    So the \(x\)-intercept is at \(\left(\frac{1}{2},0\right)\text{.}\) Plotting both intercepts, we can connect the dots with a straight line.
    üîó
    This worked, but here are some observations about why this method is not the greatest.
    • Algebra was required to solve for at least one of the intercepts.
      üîó
    • We had to plot a point with a fraction in its coordinates.
      üîó
    • We only plotted two points and they turned out to be very close to each other. So if there is even a small inaccuracy in where we marked the points or how we use a straight edge, it could result in a line that is significantly off.
      üîó
    üîó
    üîó

    Note 2.9.10.

    When a line is presented in slope-intercept form and \(b\) is an integer, our opinion is that the slope triangle method is the best choice for making its graph.
    üîó
    üîó
    üîó

    When we graph a line in point-slope form like \(y=\frac{2}{3}(x+1)+3\text{,}\) the slope triangle method is the obvious choice. We can see a point on the line, \((-1,3)\text{,}\) and the slope is apparent: \(\frac{2}{3}\text{.}\) Here is the graph:
    üîó
    Other graphing methods would take more work and miss the purpose of point-slope form. To graph a line in point-slope form, we recommend always using slope triangles.
    üîó
    üîó

    Subsection 2.9.3 Graphing Lines in Standard Form

    The line \(3x+4y=12\) is in standard form. How should we graph it? We‚Äôll review methods we learned and compare them.
    üîó

    Example 2.9.12. Building a Table of \(x\)- and \(y\)-values.

    To make a table, we could substitute \(x\) for various numbers and use algebra to find the corresponding \(y\)-values. Let‚Äôs start with \(x=-2\text{,}\) planning to move on to \(x=-1,0,1,2\text{.}\)
    üîó
    \begin{align*} 3x+4y\amp=12\\ 3(\substitute{-2})+4y\amp=12\\ -6+4y\amp=12\\ 4y\amp=12\addright{6}\\ 4y\amp=18\\ y\amp=\divideunder{18}{4}=\frac{9}{2} \end{align*}
    üîó
    The first point we found is \(\left(-2,\frac{9}{2}\right)\text{.}\) This has been a lot of calculation, and we ended up with a fraction we will have to plot. And we have to repeat this process a few more times to get more points for the table. The table method is generally not a preferred way to graph a line in standard form. So let‚Äôs look at other options.
    üîó
    üîó

    Example 2.9.13. Using intercepts.

    Next, we‚Äôll try graphing \(3x+4y=12\) using intercepts. We set up a small table to record the two intercepts:
    üîó
    \(x\)-value \(y\)-value Intercept
    \(x\)-intercept \(0\)
    \(y\)-intercept \(0\)
    We have to calculate the line‚Äôs \(x\)-intercept by substituting \(y=0\) into the equation:
    \begin{align*} 3x+4y\amp=12\\ 3x+4(\substitute{0})\amp=12\\ 3x\amp=12\\ x\amp=\divideunder{12}{3}\\ x\amp=4 \end{align*}
    üîó
    And similarly for the \(y\)-intercept:
    \begin{align*} 3x+4y\amp=12\\ 3(\substitute{0})+4y\amp=12\\ 4y\amp=12\\ y\amp=\divideunder{12}{4}\\ y\amp=3 \end{align*}
    üîó
    So the line‚Äôs \(x\)-intercept is at \((4,0)\) and its \(y\)-intercept is at \((0,3)\text{.}\) Now we can complete the table and then graph the line:
    üîó
    üîó

    Example 2.9.16. With Slope Triangles.

    We can always rearrange \(3x+4y=12\) into slope-intercept form, and then graph it with the slope triangle method:
    üîó
    \begin{align*} 3x+4y\amp=12\\ 4y\amp=12\subtractright{3x}\\ 4y\amp=-3x+12\\ y\amp=\divideunder{-3x+12}{4}\\ y\amp=-\frac{3}{4}x+3 \end{align*}
    üîó
    With the \(y\)-intercept at \((0,3)\) and slope \(-\frac{3}{4}\text{,}\) we can graph the line using slope triangles:
    üîó
    Compared with the intercepts method, the slope triangle method takes more time, but it works.
    üîó
    üîó
    Here are some observations about graphing a line equation that is in standard form:
    • The intercepts method might be the quickest approach.
      üîó
    • The intercepts method only tells us two intercepts of the line. When we need to know more information, like the line‚Äôs slope, and get a more accurate graph, we should take the time to convert the equation into slope-intercept form.
      üîó
    • When \(C=0\) in a standard form equation we cannot use the intercepts method to plot the line anyway.
      üîó
    So there‚Äôs not really a clear winner for which method is best. Using intercepts or converting to slope-intercept form and using slope triangels might be preferable depending on the situation.
    üîó
    üîó
